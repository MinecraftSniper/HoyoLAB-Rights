## 📌 维权背景

2025年11月25日，本人在HoYoLAB发布华为官方售价截图（无任何外部链接），被平台以“含广告及钓鱼网站”为由删除并禁言7天（禁言至12月3日）。  
在后续申诉中，客服处罚理由多次变更（规则#7 → 广告 → 规则#5 → 广告），最终承认“回复不准确”但仍维持处罚。  
与此同时，平台内大量包含**外链、兑换码、邀请码**的真实广告却长期存活，构成明显双重标准。
